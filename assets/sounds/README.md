# Animation Sounds

This directory contains sound effects for the special Mohamed animations.

## Required Sound Files

Add the following sound files (in .wav, .mp3, or .m4a format) to this directory:

- `agree.wav` - Sound for the Agree Gesture animation (F key)
- `dance.wav` - Sound for the All Night Dance animation (G key)
- `boom.wav` - Sound for the Boom Dance animation (H key)
- `boxing.wav` - Sound for the Boxing Practice animation (J key)
- `dead.wav` - Sound for the Dead animation (K key)
- `skill.wav` - Sound for the Skill animation (L key)

## Sound Format

- Supported formats: WAV, MP3, M4A
- Recommended: Short sound effects (1-3 seconds)
- The sounds will be played as positional audio, so other players will hear them from your character's position

## Keybinds

- **F** - Agree Gesture
- **G** - All Night Dance
- **H** - Boom Dance
- **J** - Boxing Practice
- **K** - Dead
- **L** - Skill

## Notes

- If a sound file is missing, the animation will still play but without sound
- Sounds are synchronized across all players in the multiplayer session
- Each player hears sounds from other players' positions using 3D positional audio
